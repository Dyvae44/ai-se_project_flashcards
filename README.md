#Flashcard App

## Description

My first project in TripleTen's AI-Assisted Software Engineering program. It includes decks of flashcards, each of which can be viewed in a carousel.

# Technologies Used

- HTML
- CSS
- Javascript

## How to use

1. Open idenx.html in your web browser
2. Click through flashcards to study
3. Memorize and test your knowledge!

## Features

- Interactive flascards
- Clean, user-friendly interface
- Easy navigation between cards
